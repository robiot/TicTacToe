# TicTacToe
Basic 2 Player Tic Tac Toe that I made in python cause "why not"

You can play against another human or against the computer if you dare :)

Looks like:
```
  A  |  B  |  C  

     |     |
  #  |  #  |  #    | 1
_____|_____|_____
     |     |
  #  |  #  |  #    | 2
_____|_____|_____
     |     |
  #  |  #  |  #    | 3
_____|_____|_____
     |     |

Player1 (X) > 
```
